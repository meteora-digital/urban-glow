# Urban Glow Theme for VS Code

A vibrant, city lights colour palette for Visual Studio Code, designed to bring a unique and colorful urban glow aesthetic to your coding environment.

## Features
- Carefully crafted color palette for optimal readability and style
- Enhanced syntax highlighting for a wide range of languages
- Custom UI colors for a cohesive look
- Inspired by vibrant city lights and urban nightscapes

## Installation
1. Open the Extensions sidebar in VS Code (`Ctrl+Shift+X` or `Cmd+Shift+X`)
2. Search for `Urban Glow`
3. Click **Install**
4. Select the theme via `Preferences: Color Theme`

## Usage
After installation, open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`), type `Color Theme`, and select **Urban Glow**.
